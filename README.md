# Modelo para traballos FIC
## Instrucións
## Adaptar o modelo

Cambiar autor e título tanto nos apartados de metadatos, como no título mostrado na páxina.

## Compilar

Compilar con BibLaTeX e XeLaTeX, para que inclúa as referencias.
